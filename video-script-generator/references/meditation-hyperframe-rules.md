# 冥想课程 Hyperframe 字幕改造规则

## 改造目标

将冥想课程脚本改造为 hyperframe 无人出镜短视频的字幕格式。

---

## 核心原则

### 跟练/实践类（引导冥想型）

**跟练部分（引导语）**：尽可能完整保留引导词，一个呼吸周期 ≈ 一行，保持流畅的引导节奏。

**开场白/结束语**：可以精简口水话，删除课程预告（如"下节课见""祝你美好的一天"）。

**改造规则：**
1. 一个呼吸周期 = 一行字幕
2. 「名词 → 方向」的节奏
3. 删除旁白词（「很好」「接下来」「Ok」「太棒了」「非常棒」）
4. 删除图说和引用注释
5. 删除开课仪式和结束语中的课程预告（「下节课见」「明天见」）
6. 删除「我会完成14天冥想」等具体课程承诺，改为通用意愿语
7. 删除结果导向的结尾想象
8. 保留完整的自我对话意愿语（简化格式）

**示例：**
```
原文：很好，接着感受左脚的脚掌。脚踝。小腿。膝盖。大腿。我们不是在搜寻什么特别的感受，只是把注意力放在身体的不同部位。

改造后：
左脚
脚掌
脚踝
小腿
膝盖
大腿

不是在搜寻特别的感受
只是感受它们的存在
```

---

### 讲解类（知识讲解型）

**改造规则：**
1. 一个论点 = 一行字幕
2. 删除所有例子和引用（只保留结论）
3. 删除「首先、其次、最后」结构，改成独立论点
4. 删除「如果大家对xxx感兴趣可以搜索xxx」等延伸阅读
5. 删除课程结构介绍（「讲解+跟练+答疑+实践」「每四天一个小单元」）
6. 删除课程承诺（「这门课你一定要练到第二周」「每天十分钟不到一个月就能看到效果」）
7. 删除「预告下节课是xxx」「下节课我们讲」等课程预告
8. 把个人经历和故事替换成通用的知识点
9. 删除所有图说和 OCR 注释

**示例：**
```
原文：焦虑、抑郁和专注力差等问题越来越普遍。为什么在文明进步的同时，人类的身心健康却反而退步了呢？其中一个主要原因是我们的大脑进化程度没有跟上时代变化的速度。

改造后：
文明进步
身心却在退步

原因：
大脑进化速度
跟不上时代变化
```

## 必须删除的内容

| 类型 | 具体内容 |
|---|---|
| 课程预告 | 「下节课是xxx」「下节课我们讲」「明天见」「讲解课上见」 |
| 课程承诺 | 「我会完成14天冥想」「这门课你一定要练到第二周」 |
| 结果想象 | 「想象完成后的成就感」「想象14天后身心状态的改善」 |
| 旁白词 | 「很好」「太棒了」「Ok」「非常棒」「接下来」「动作很简单」 |
| 自我对话 | 简化保留，删除「我念一句你跟念一句」引导语，保留意愿语正文 |
| 旁白词 | 「很好」「太棒了」「Ok」「非常棒」「接下来」「动作很简单」 |
| 图说注释 | `<!-- 这是一张图片，ocr 内容为：xxx -->` |
| 课程结构 | 「讲解+跟练+答疑+实践」「每四天一个小单元」「14天主体完成后」 |
| 延伸阅读 | 「如果对这个话题感兴趣可以搜索xxx」「文献参考在简介中」 |
| 铺垫解释 | 「有时候你可能会感觉很难坐得住，不愿意坐下来，其中一个原因可能是...」 |

## 保留的内容

- 核心引导词（跟练类）
- 核心论点（讲解类）
- 身体部位名称（跟练类）
- 关键知识点（讲解类）
- 开放性问题（「身体是冷还是热？」）

---

## 文件命名

**原则：名字不暴露课程结构，不用"D1""序号""先导""讲解"等词汇，用意象词/动词/单字。**

```
序号-新标题.hyperframe.md
```

**新标题命名规则：**
- 跟练/实践类：用动词或意象词（1-2个字），如「归零」「向内走」「沉」「静坐」
- 讲解类：用短语或意象词（2-4个字），如「与脑共处」「呼吸的功课」「体感」
- 禁止出现：D1/D2/序号/先导片/跟练/实践/讲解/第X天/14天等暴露课程结构的词
- 可以用：一句话、一个意象、一个动作、一个状态


---

## 改造流程

1. 读原始脚本
2. 判断类型（跟练/实践类 → 提取引导词；讲解类 → 压缩论点）
3. 应用改造规则逐行改写
4. 删除所有「必须删除的内容」
5. 输出到 `hyperframe/` 目录
6. 复核一遍确认无多余内容
