# 微信公众号正文获取

微信公众号经常返回环境验证页。遇到 `mp.weixin.qq.com` 链接必须先过正文校验，再进入提炼。

## 正文校验

1. 检查标题、作者、发布时间、正文开头和正文长度。
2. 页面出现"环境异常""完成验证后即可继续访问""去验证"等字样时，判定为验证页。
3. 验证页不能进入总结或生成 slides。
4. 尝试移动端 User-Agent、微信内置浏览器 User-Agent、原始链接和可读代理服务。
5. 拿到 HTML 后，从 `js_content` 抽正文，并记录链接、标题、作者、发布时间和正文字符数。
6. 本地已有 `wechat-article.md`、旧 HTML 或转写稿时，必须核对 Source URL 或标题。链接不一致时禁止使用。
7. 多种方式都拿不到正文时，要求用户粘贴正文或提供导出的 md、html、pdf。

## 进入提炼前

用一句话说明材料依据，例如：已从链接抓到正文，标题为某某，正文约多少字。
